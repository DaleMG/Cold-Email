# Cold Email App

Send cold emails directly from your Gmail account.

## Features

* Google sign-in
* Manage contacts
* Single email template
* Batch sending
* Track status (`not_sent` / `sent`)

## Tech Stack

Next.js, Auth.js (Google OAuth), Supabase (Database), Gmail API

## How It Works

1. Sign in
2. Add contacts
3. Write template
4. Send emails

## Limitations
* Application is in testing mode/unverified (Google Cloud), so only authorized test users can access it
* Single template only (for now)

## Future Improvements
* Multiple templates
* Email sending limits
* Attachments support
* Improved UI/UX feedback
